\section{Требования к внешним интерфейсам }
При запуске ПО «Монитор» должно появляться основное окно программы. В нём должны присутствовать следующие элементы:
\begin{itemize}
\itemТаблица со списком процессов и данными о них (имя процесса, имя рабочей станции, имя запустившего процесс пользователя, затрат CPU и оперативной памяти в процентах от общего числа)
\itemТаблица со списком рабочих станций и данных о них (имя рабочей станции, загруженность CPU и оперативной памяти в процентах от общего числа и килобайтах)
\itemКнопка «Обновить данные»
\itemКнопка «Настройки», по нажатию кнопки «Настройки» должно появляться окно настройки
\end{itemize}

Окно настройки должно содержать в себе следующие элементы:
\begin{itemize}
\itemполе для ввода времени авто обновления информации
\itemкнопки «ОК» и «Отмена»
\itemсписок рабочих станций, на с которых на данный момент можно собрать данные, с элементами checkbox рядом с каждым именем
\itemСписок столбцов данных в таблице процессов с элементами checkbox рядом с каждым именем
\end{itemize}
